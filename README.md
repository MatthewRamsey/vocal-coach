# Vocal Coach

A private, browser-based singing practice app focused on feedback that can be measured reliably.

## Product priorities

1. Accurate target-note matching with confidence-aware scoring.
2. Guided five-note practice with clear, actionable feedback.
3. Attempt summaries showing median pitch error and time in tune.
4. Later: melody timing, recording/playback, history, and validated tone-quality exercises.

The app deliberately does not claim to infer confidence, pronunciation, breath support, or voice type from generic microphone statistics.

## Run and verify

```bash
npm run dev
npm test
npm run test:e2e
npm run build
```
